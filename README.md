# CSE2425 Embedded Software 2025

Repository for TU Delft CSE2425 Embedded Software lab assignments.

## Repository layout

- `robot/` RP2040 robot firmware (current work and snapshots)
- `robot-templates/` starter templates for robot assignments
- `debug/` standalone debug exercise
- `hashmap/` standalone hashmap exercise/project
- `LICENSE` license file

## Quick start

- Main robot code: see `robot/README.md`
- Build/flash targets are in `robot/Makefile`
