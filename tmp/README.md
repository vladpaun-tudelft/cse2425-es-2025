# CSE2425 Robot Templates
This repository contains template projects for the CSE2425 robot lob for the following frameworks/languages. The templates all demonstrate how to blink the on-board LED of the provided STM32F401CCU6.
- [Arduino](https://arduino-pico.readthedocs.io/en/stable/index.html): `arduino`
- [C SDK](https://datasheets.raspberrypi.com/pico/raspberry-pi-pico-c-sdk.pdf): `pico_sdk`
- [Registers](https://datasheets.raspberrypi.com/rp2040/rp2040-datasheet.pdf): `pico_sdk`
- [Rust](https://docs.rust-embedded.org/book/intro/index.html): `rust`

## Prerequisites

In order to compile and upload the provided code (with the exception of Rust; please see the Rust-specific instructions), you must first
1. [Install VSCode](https://code.visualstudio.com/)
2. [Use the official RP Pico extension](https://marketplace.visualstudio.com/items?itemName=raspberry-pi.raspberry-pi-pico)
3. [Or any other editor using picotool](https://datasheets.raspberrypi.com/pico/getting-started-with-pico.pdf)
